# Skill Evaluation Framework

This project integrates the `skillscan` and `llmguard` testing tools, using the `test_my_skill.py` script to seamlessly perform security testing.

## Directory Structure
- `test_my_skill.py`: The main test script that calls the relevant skillscan modules to run the scan.
- `skillscan/`: Contains the specific code logic.
  - `SecurityCheck/`: Core security classes (e.g., `SkillSecurityScan.py` and `LLMGuard.py`).
  - `requirements.txt`: The dependency list for this project.
- `my_reports/` (auto-generated after execution): Saves the JSON report files generated by the tests.

## Dependency Installation

Before running, you must install the relevant Python dependencies. Specifically, `skillscan` and `LLMGuard` rely on specific packages. You can run the following command to install them:

```bash
pip install -r skillscan/requirements.txt
```
The minimum required libraries are:
- `requests`
- `llm-guard`
- `pyyaml`

*Note: If the terminal outputs "LLMGuard not found" during execution, it means the `llm-guard` component is missing or the environment variables are not set. Please ensure the dependencies are installed correctly.*

## Execution Flow

1. **Prepare the skill directory to be scanned**: Prepare the target system or directory in advance.
2. **Run the test script**: Execute the following command, passing the path of the skill directory to be scanned.
   ```bash
   python test_my_skill.py <path_to_skill_directory>
   ```
   *You can pass multiple directories or provide a root directory containing multiple skills.*
3. **Security Scan (SkillSecurityScan)**: The script will call `skillscan.SecurityCheck.SkillSecurityScan` to perform a static code scan.
4. **LLMGuard Analysis**: The script reads all `*.md` files under the target directory and uses LLMGuard to scan for prompt leakage and security compliance.
5. **Generate Reports**: Once execution is complete, individual scan results and a summary report (`final_skill_scan_summary.json`) can be found in the `my_reports/` directory.
